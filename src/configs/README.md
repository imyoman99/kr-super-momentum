# Config

## 역할 (Role)
- 전략 실행에 필요한 모든 파라미터의 공급원
## 담당 (Owner)
박재진 (Quant), 양태영 (Quant/Dev), 오은원 (Dev/Vis)

## 주요 내용 (What)
### 전략 설정
- 포트폴리오 구성 파라미터
  - 최대 보유 종목 수
  - 종목별 비중 방식 (equal / weighted)
- 리밸런싱 설정
  - 리밸런싱 주기 (monthly / Quarterly)
  - 리밸런싱 트리거 조건

### 모멘텀 설정
- 사용 지표
- 지표별 파라미터 값

### 리스크 설정
- 최대 손실 한도
- 포지션 사이징 규칙
- 손절/익절 기준

## 사용 흐름 (How)
- 전략 담당자는 config만 수정하여 전략을 조정
- 백테스터는 config를 로딩하여 전략 로직에 반영
