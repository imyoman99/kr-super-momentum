# Strategy

## 역할 (Role)
- 투자 전략의 전체 구조 정의
- 진입/청산 및 포트폴리오 구성 및 리밸런싱 규칙까지 포함

## 담당 (Owner)
- 박재진 (Quant), 오은원 (Dev/Vis) / 보조 : 양태영 (Quant/Dev)

## 주요 내용 (What)
- 진입 / 청산 조건
- 포트폴리오 구성 방식
  - 종목 수 제한
  - 동일비중 / 가중비중
- 리밸런싱 규칙
  - 주기 기반 (월간 / 분기)
  - 이벤트 기반 (신호 변화)
- 전략 흐름을 **사람의 언어로 명확히 문서화** 및 코드로 구현

## 사용 흐름 (How)
1. 전략 담당자가 문서로 전략을 정의
2. 백테스터 담당자가 이를 코드로 구현
3. config 파일을 통해 파라미터화
